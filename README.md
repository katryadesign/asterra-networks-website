# Asterra Networks Website

Marketing/company website for Asterra Networks. Built with [Next.js](https://nextjs.org) (App Router), TypeScript, and Tailwind CSS.

> Skeleton scaffold — site content is being filled in.

## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it. Edit `app/page.tsx` and the page hot-reloads.

## Project Structure

```
app/           # App Router pages, layouts, and global styles
public/        # Static assets
next.config.ts # Next.js configuration
```

## Deployment

Hosted on **Vercel** with Git-based autodeploy:

- **Production:** https://asterra-networks-website.vercel.app
- Every push to `main` → a production deploy.
- Every pull request → its own preview URL.

No manual deploy step is needed — push to GitHub and Vercel builds and ships automatically.
